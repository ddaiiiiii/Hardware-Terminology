Input Device

An Input Device turns physical events into zeros and ones. Allows user to interact and control the computer. For instance, a keyboard, mice, mic....

Output Device

An Output Device turns zeros and ones from the computer to something physical. For instance, haptic feedback, printer, projector....

Central Processing Unit (CPU)

A Central Processing Unit (CPU) is an electronic circuitry that decodes and executes instructions from software programs.It also performs arithmetic calculations which controls and supports the timing of each operation.

Arithmetic Logic Unit (ALU)

An Arithmetic Logic Unit (ALU) is an important component of the CPU. It processes arithmetic and logic operations that need to be done on instruction words.

Control Unit

A Control Unit is also a component of a computer's central processing unit (CPU) that directs the operation of the processor. It tells the computer's memory, arithmetic logic unit and input and output devices how to respond to the instructions that have been sent to the processor.

Random Access Memory (RAM)

A Random Access Memory (RAM) is a form of computer memory that can be read and changed in any order, typically used to store working data and machine code. A random-access memory device allows data items to be read or written in almost the same amount of time irrespective of the physical location of data inside the memory.

Optical Memory

An Optical Memory is a disk that is formed from polycarbonate. The data on the disk (CD) is stored on the disc as a series of tiny, almost microscopic pits and lands.

Magnetic Memory

A Magnetic Memory is the storage of data on a magnetized medium. Magnetic storage uses different patterns of magnetisation in a magnetizable material to store data and is a form of non-volatile memory

Flash Memory

A Flash Memory is a long-life and non-volatile storage chip that is widely used in embedded systems. It can keep stored data and information even when the power is off and can be electrically erased and reprogrammed.

Logic Gates

A Logic Gate is the basic building blocks of any digital system. It is an electronic circuit having one or more than one input and only one output. The relationship between the input and the output is based on a certain logic. Based on this, logic gates are named as AND gate, OR gate, NOT gate etc.
